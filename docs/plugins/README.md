# StreamSharp Plugins Documentation

This folder contains documentation for StreamSharp plugins.

## Available Plugins

- **[StreamSharp.PostgreSQL](./PostgreSQL/README.md)** - Event Store and Read Model implementation using Entity Framework Core and PostgreSQL
- **[StreamSharp.UI](./UI/README.md)** - Vue 3 + TypeScript + TailwindCSS frontend SPA embedded in the server

## Plugin Development

Each plugin includes:
- Architecture and design patterns
- Development setup and build instructions
- Database migrations guide (for data plugins)
- API endpoints and services
- Testing strategies

## General Plugin Information

For general plugin development guidelines, see the main [agents.md](../../agents.md#plugin-system) documentation.
